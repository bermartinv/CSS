>Referencia > Propiedades > filter

filter: Es una herramienta CSS que permite añadir efectos visuales de una imagen como si estuvieramos haciendolo con un editor de imagenes.

Valores

  • blur()
  • brightness()
  • contrast()
  • drop-shadow()
  • grayscale()
  • hue-rotate()
  • invert()
  • opacity()
  • saturate()
  • sepia()
  • url() - para aplicarfiltros SVG
  • custom()

Comentar que se pueden aplicar varios estilos distintos a la vez.
