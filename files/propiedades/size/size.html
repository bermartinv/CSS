>Referencia > Propiedades > width / height

width / height : Como indica su traducción de estas palabras significan ancho y alto.

1. Las medidas que podemos usar son relativas o absolutas.

2. Una de las cosas más importantes a tener en cuenta cuando usemos estas características en nuestro CSS, es que no es aplicable a elementos que sean de tipo inline. Un ejemplo de este elemento podría ser span, el cual si le damos altura y anchura el navegador lo ignorará.

La unidad de medida tendrá que ir junto con la cantidad de medida, sin ningún espacio entre ellas, para que se pueda ejecutar.

4. Habrá que tener en cuenta en el ancho y alto que queramos, el padding y el margin que tenga nuestro elemento ya que se sumará a la medida que hayamos especificado a su height y width.

5. Uno de los posibles trucos que podemos hacer para conseguir que un elemento que sea inline podamos dotarle de height y width es dandole la siguiente característica CSS display:inline-block;, de esta manera conseguiremos poder definir sus características de height y width.
