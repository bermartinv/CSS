>Referencia > Unidades > Absolutas

cm : Medida en centímetros.

Equivalencia

1cm = 96px/2.54

mm : Medida en milímetros.

Equivalencia

1mm = 1/10 de 1cm (una décima parte de un centímetro)

Q : Un cuarto de milímetro.

Equivalencia

1q = 1/40 de 1cm (1q es una cuarenta parte de 1 centímetro)

in : Esta medida se refiere a pulgadas.

Equivalencia

1in = 2.54cm = 96px

pc : Medida en picas.

Equivalencia

1pc = 1/6th de 1in (1 pica es una sexta parte de una pulgada)

pt : Medida en points.

Equivalencia

1pt = 1/72 de 1in (un point es 72 parte de una pulgada)

px : Medida en pixels.

Equivalencia

1px = 1/96th of 1in (un pixel es la 96 parte de una pulgada)
