>Referencia > Selectores>Relacionales

Descendente (E F) : El seleceltor se dirige a todo elemento que es descendente de él. Es este caso sería todos los elementos F que desciende de E.

Ejemplo

See the Pen descendente by Bernardo Martin (@bermartinv) on CodePen.

Hijo (E>F) : Se selecciona un elemento F que es hijo directo de E.

Ejemplo

See the Pen hijo by Bernardo Martin (@bermartinv) on CodePen.

Hermanos adyacentes (E + F) : El selector selecciona todos los elementos de tipo F en el que este aparece inmediatamente despues de E y además F debe tener el mismo padre que E

Ejemplo

See the Pen hermano descendente by Bernardo Martin (@bermartinv) on CodePen.

Hermano general (E ~ F):Seleccionaremos el elemento F que siga a in E, siempre y cuando ambos compartan el mismo padre directo.

Ejemplo

See the Pen hermano general by Bernardo Martin (@bermartinv) on CodePen.
