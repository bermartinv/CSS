>Referencia > Selectores>Basicos

Tipo Selector : Se indica el tipo de etiqueta a que se le va a definir las características con CSS. Además se puede declarar distintas etiquetas a la vez separadas entre "," si van a tener las mismas características.

Ejemplo

See the Pen tipo selector by Bernardo Martin (@bermartinv) on CodePen.

Selector class : Añadimos la etiqueta class y un nombre a aquellos elementos que le queramos dar cierta característica. El nombre del class lo podemos encontrar en distintos elementos y puede repetirse entre nuestro código.En nuestro codigo CSS lo definimos con un . y su nombre (.name)

Ejemplo

See the Pen class name by Bernardo Martin (@bermartinv) on CodePen.

Selector id : El selector id una vez definido con un nombre, este solo puede aparecer una vez en el código que estamos trabajando.Nos referimos a este id en nuestro código CSS de la forma #name

Ejemplo

See the Pen selector id by Bernardo Martin (@bermartinv) on CodePen.
